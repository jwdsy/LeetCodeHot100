## 项目简介

本项目是 LeetCode 热题 100 的 Java 刷题仓库，所有解法按题目类型分类组织，每道题包含：

- ✅ 完整题目描述
- ✅ 解题思路分析
- ✅ 最优 Java 实现
- ✅ 时间/空间复杂度说明
- ✅ 测试用例

## 核心算法

- **哈希表**: 两数之和、字母异位词分组、最长连续序列
- **双指针**: 移动零、三数之和、接雨水
- **滑动窗口**: 无重复字符的最长子串、最小覆盖子串
- **链表**: 反转链表、环形链表、合并K个升序链表
- **二叉树**: 层序遍历、最近公共祖先、最大路径和
- **动态规划**: 最大子数组和、打家劫舍、编辑距离
- **回溯**: 全排列、N皇后、括号生成
- **二分查找**: 旋转数组搜索、中位数
- **贪心**: 股票买卖、跳跃游戏

## 刷题建议

1. **按分类刷**: 同类型题目放在一起，加深对同一解法的理解
2. **先看思路**: 尝试自己写，写不出来再看注释中的思路
3. **难度递进**: 先刷 Easy，再攻 Medium，最后挑战 Hard
4. **每日复盘**: 每天刷 3-5 题，并回顾昨天的题目
